# Course 4: GenAI Engineering Labs

## Lab Structure

Each lab includes:
- Databricks notebook (`.py` export)
- Sovereign AI Stack equivalent (Rust)
- Hands-on exercises

## Labs by Week

### Week 1: LLM & Prompt Engineering
- `lab_tokenizer.py`: Build a BPE Tokenizer
- `lab_prompts.py`: Prompt Template Engine

### Week 2: Vector Search & RAG
- `lab_embeddings.py`: Generate and Store Embeddings
- `lab_rag.py`: End-to-End RAG Pipeline

### Week 3: Fine-Tuning & Production
- `lab_finetuning.py`: Fine-Tune with LoRA/QLoRA
- `lab_production.py`: Production Deployment
