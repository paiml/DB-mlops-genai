## Key Concepts

### Prompt Patterns
- Zero-shot: direct instruction
- Few-shot: examples in prompt
- Chain-of-thought: step-by-step reasoning
- Role-based: system/user/assistant

### Structured Output
- JSON mode for parseable responses
- Schema validation
- Type-safe deserialization
