# Databricks Examples

Databricks notebook examples exported as Python files.

## Structure

```
databricks/
  mlflow/           # MLflow tracking examples
  feature-store/    # Feature Store examples
  model-serving/    # Model Serving examples
  vector-search/    # Vector Search examples
  rag/              # RAG pipeline examples
```

## Usage

Import these notebooks into your Databricks workspace:
1. Go to Workspace
2. Click Import
3. Select the `.py` file
